# Matching game

Just a small matching game that I made for my son

## how to run

- get the dependencies by running

```
npm i
```

- from cli run:

```
npm run dev
```

## Created with:

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Zustand](https://github.com/pmndrs/zustand)
- [Lorem Picsum](https://picsum.photos/)
